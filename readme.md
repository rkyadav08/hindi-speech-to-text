# Hindi Speech-to-Text
 
Fine-tuned Whisper-small model for Hindi transcription.
 
## Features
- Upload audio files (WAV, MP3, FLAC, OGG, M4A)
- Record from microphone
- Dictation mode — record multiple clips that append into a single document
- Copy button to paste directly into Word/Google Docs
